# Grocerra 🛒

A real-time shared grocery list app built for Indian households.

## Features
- Family sync — multiple devices share one live list via Firebase
- Add items with count, weight, variety, and brand/company
- Pre-loaded Indian grocery database (120+ items, Hindi + English)
- Frequently bought items for quick add
- PDF export with customer details
- Light / Dark theme
- Offline-first — changes sync when connection returns
- Secure family rooms with ID + password

## Tech Stack
HTML · CSS · JavaScript · Firebase Realtime Database · jsPDF
